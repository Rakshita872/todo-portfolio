todo-portfolio/
│

---

## ✨ Features

- ✅ Add new tasks instantly
- ✅ Mark tasks as complete / incomplete
- ✅ Delete individual tasks
- ✅ Filter tasks - All / Active / Done
- ✅ Progress Bar showing completion %
- ✅ Clear all completed tasks at once
- ✅ Tasks saved in localStorage
- ✅ Fully responsive on mobile
- ✅ Smooth animations on task entry
- ✅ Press Enter key to add a task

## 🗂️ File Breakdown
index.HTML
- Page structure and layout
- Links to style.css and script.js
- Contains the input feild, filter buttons, progress bar, and task list

style.css
- Dark theme with CSS variables
- Custom checkbox Styling
- Slide-in animation for tasks
- Fully responsive with mobile breakpoints

script.js
- addTask()-- adds a new tasks
- toggleTask(id)-- marks task done/undone
- deleteTask(id)-- removes a tasks
- clearDone()-- removes all completed tasks
- setFilter()-- filters by All/Active/Done
- render()-- updates the UI
- localStorage for data persistance
